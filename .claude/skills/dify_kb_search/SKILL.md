# 电工知识库检索技能

## 触发条件

仅当用户输入包含以下关键词之一时触发：

- **科目一** / **科目二** / **科目三** / **科目四**

### 1. 直接读本地文件，不走 Dify API

### 2. 输出全文

- 直接输出文件完整内容（markdown 格式）
